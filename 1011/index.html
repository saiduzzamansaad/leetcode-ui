Problem Statement

A conveyor belt has packages that must be shipped from one port to another within D days. The ith package has a weight of weights[i]. Each day, we load the ship with packages on the conveyor belt (in the order given). We may not load more weight than the maximum weight capacity of the ship.

Return the least weight capacity of the ship that will result in all the packages being shipped within D days.

Binary Search Algorithm

The calculator uses an efficient binary search approach to solve this classic optimization problem:

  1. Initialize search range: Minimum capacity is the heaviest package (must ship each package), maximum is the sum of all weights (could ship all in one day)
  2. Binary search: Test middle values in this range to find the optimal capacity
  3. Feasibility check: For each test capacity, simulate the shipping process to check if packages can be shipped within D days
  4. Adjust range: If feasible, try smaller capacities; if not, try larger capacities
  5. Converge: Continue until the minimum viable capacity is found

Complexity Analysis

Time: O(n log n) - Binary search is O(log n) and each feasibility check is O(n)
Space: O(1) - Uses constant extra space

Algorithm Implementation 

function shipWithinDays(weights, days) {
    let left = Math.max(...weights);
    let right = weights.reduce((a, b) => a + b, 0);
    
    while (left < right) {
        const mid = Math.floor((left + right) / 2);
        let current = 0, needed = 1;
        
        for (const weight of weights) {
            if (current + weight > mid) {
                needed++;
                current = 0;
            }
            current += weight;
        }
        
        if (needed <= days) right = mid;
        else left = mid + 1;
    }
    
    return left;
}

This optimization problem lies at the heart of efficient logistics operations. Finding the minimum ship capacity that meets delivery deadlines allows companies to:

  • Right-size their transportation fleet (avoiding over-capacity)
  • Reduce fuel costs and carbon emissions
  • Improve delivery reliability and customer satisfaction
  • Make data-driven decisions about shipping schedules

The binary search approach efficiently narrows down the optimal capacity by:

  1. Setting the initial search range between the heaviest package and the total sum of all weights
  2. Testing the midpoint capacity to see if it's feasible to ship within D days
  3. Adjusting the search range based on the feasibility result
  4. Repeating until the minimum feasible capacity is found

This approach is dramatically faster than linear search, especially for large datasets.

While powerful, this solution makes several assumptions:

  • Packages must be shipped in the given order (no reordering)
  • All packages must be shipped (no partial shipments)
  • Ship capacity is constant across all days
  • No consideration for package dimensions (only weight)
  • Single shipping route (no multiple destinations)

Real-world implementations often extend this basic algorithm to handle these additional constraints.
