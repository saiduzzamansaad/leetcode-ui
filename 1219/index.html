Mining Tutorial

Understanding the Mine

The mine is represented as a grid where each cell contains gold (positive number) or is empty (0). Your goal is to find a path that collects the maximum amount of gold.

Movement Rules

You can move in 4 directions: up, down, left, or right. You cannot move diagonally. The path must be continuous and cannot revisit any cell.

Optimal Path

The algorithm explores all possible paths using backtracking to find the one with maximum gold. It marks visited cells to avoid cycles and backtracks when no more moves are possible.
