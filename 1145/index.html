Binary Tree Coloring Game

Master the strategy of this two-player game where coloring nodes determines the winner

Interactive Demo Learn Examples

Problem Statement

Two players take turns coloring nodes in a binary tree. The first player (Red) colors any node, then the second player (Blue) colors any uncolored node. Players then alternately color uncolored neighbors of their colored nodes.

As the second player (Blue), can you choose an initial node that guarantees a win?

Key Insight

The second player can win by controlling >50% of nodes through strategic partitioning of the tree.

Solution Approach

  1. Find the node colored by first player (x)
  2. Calculate sizes of its left, right, and parent subtrees
  3. If any subtree has > n/2 nodes, Blue can win by selecting the adjacent node
  4. Otherwise, Red can maintain control and win
Time Complexity: O(n)
Space Complexity: O(h) [h = tree height]

Winning Strategies

Parent Control

If parent subtree is largest, select parent node to control all nodes above x.

Left Dominance

If left subtree is largest, select left child to control that entire branch.

Right Dominance

If right subtree is largest, select right child to control that entire branch.
